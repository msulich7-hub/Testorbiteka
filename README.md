# Testorbiteka

Repozytorium robocze pod pomysły i dokumentację związaną z platformą **[Orbiteus](https://github.com/orbiteus/orbiteus)** (silnik ERP/CRM / aplikacji biznesowych).

## Dokumenty

| Dokument | Opis |
|----------|------|
| [Orbiteus CRM — megadesign](docs/crm-megadesign.md) — [**podgląd na Pages**](https://msulich7-hub.github.io/Testorbiteka/crm-megadesign/) | Synteza researchu wielu CRM, backlog P0/P1/P2, role, mapowanie modułów. |

### Podgląd na GitHub Pages

Po włączeniu Pages (**Settings → Pages → Branch: `main`, folder `/docs`**) dokumentacja jest dostępna pod  
`https://msulich7-hub.github.io/Testorbiteka/` (start) oraz `/crm-megadesign/` (megadesign). Jeśli zmienisz nazwę repozytorium, zaktualizuj `baseurl` w [`docs/_config.yml`](docs/_config.yml).

Upstream Orbiteus klonuj z oficjalnego repozytorium; tutaj możesz trzymać briefy produktowe i notatki pod własny fork lub moduły.
